# Contributing

Thanks for helping improve `repo-to-content`.

## Development

Use Node.js 20 or newer.

```bash
npm install
npm run release:check
```

## Pull Requests

- Keep changes focused and include tests or fixture updates for behavior changes.
- Run `npm run release:check` before opening a PR.
- Keep generated launch content deterministic and grounded in fixture data.

## Issues

Please include the command, relevant fixture shape, and expected versus actual output.
