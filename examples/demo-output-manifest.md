# Fixture Demo Output Manifest

Running `bash demo/run-fixture-content.sh` produces a temporary launch pack with
the following files:

| File | Purpose |
| --- | --- |
| `posts.md` | Short social draft material based on README and package metadata. |
| `video-script.md` | A compact script for showing the repo and proof files. |
| `launch-notes.md` | Evidence-backed launch notes checked by the CLI. |
| `evidence.json` | Machine-readable claims and their local sources. |

Use this manifest when reviewing demo output in a PR, video brief, or release
prep checklist.
